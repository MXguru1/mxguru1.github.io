## What is Sovereign Hive?

Sovereign Hive is a local-first, multi-agent AI security and operations platform that runs an 8-agent adversarial swarm on a single consumer GPU (RTX 5070, 12GB VRAM). No cloud dependency. No API keys. Full air-gap capable.

Built on 11 sub-projects with frozen-contract architecture, defense-in-depth containerization, and kernel-level isolation via eBPF tripwires.
